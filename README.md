## Password Strength Checker

A Python GUI application built using Tkinter that checks password strength in real time and provides visual feedback.

## Features:
- Real-time password strength detection
- Strength levels: Weak / Medium / Strong
- Color-based visual feedback
- Modular code structure (GUI + logic separated)

## Tech Stack:
- Python
- Tkinter
